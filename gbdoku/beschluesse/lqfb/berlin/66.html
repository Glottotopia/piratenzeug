i66: Einführung einer "Meine Kontakte auf dieses Thema hinweisen"-Funktion
Initiative i66: Einführung einer "Meine Kontakte auf dieses Thema hinweisen"-Funktion
 Ja: 35 (83%) · Enthaltung: 7 · Nein: 7 (17%) · Angenommen
Letzter Entwurf vom 06.01.2010 um 12:49:24 Uhr

Einführung eines Buttons "Meine Kontakte auf dieses Thema hinweisen". Es folgt ein Auswahlfeld in dem man entweder allen Kontakten oder nur ausgewählten Kontakten den Hinweis auf ein Thema zukommen lassen kann.

Die auf das Thema hingewiesenen Kontakte erhalten dann auf ihrer Startseite eine Nachricht: "XY weist Dich auf Initiative AB" hin, mit einem Link um sich die Initiative direkt anzusehen.

Gleichzeitig soll jeder Nutzer die Möglichkeit haben eine Liste zu erstellen von welchen Nutzern er keine Hinweise wünscht, um Spam zu verhindern.

Nutzer die einen Hinweis gut finden sollen über eine "Meine Kontakte ebenfalls auf dieses Thema hinweisen"-Funktion die Möglichkeit haben einen Hinweis direkt weiterzuleiten.
