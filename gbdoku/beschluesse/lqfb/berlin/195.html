Letzter Entwurf vom 17.01.2010 um 19:03:25 Uhr

Ich hatte schon eine Initiative "Bündnisse" gestartet, die auch eine Angabe politischer Ziele für einzelne Nutzer ermöglicht. Allerdings dies mit weitergehenden Möglichkeiten kombiniert:

  • Zielangabe für Bündnisse
  • Jeder Nutzer kann Bündnisse gründen.
  • Somit können auch Ich-Bündnisse gegründet werden.

Die Bündnisse haben somit zusätzlich den Vorteil, dass sie parallel als Navigationsinstrument dienen können, um Piraten zu finden, an die man delegieren möchte. Keiner möchte sich durch 300 Nutzerbeschreibungen von politischen Zielen durchwühlen.

Außerdem kann auch schon jetzt jeder einen freien Text auf seine Nutzerseite stellen. Dieser Text kann (und sollte eigentlich) für die Darstellung der eigenen politischen Ziele / Normen / Vorstellungen genutzt werden.
