Das ist ein Meinungsbild, kein Antrag!

Idee

Firmen sind verpflichtet Software die älter als 10 Jahre ist und nicht mehr vom Hersteller supportet wird, soweit möglich, mitsamt Quellcode unter einer Open Source Lizens freizugeben. Die Lizens muss eine von der OSI bestätigte Open Source Lizens sein → http://www.opensource.org/licenses/alphabetical

Begründung

Software auf die diese Bedinungen zutrifft, ist bereits kommerziell voll verwertet. Jedoch sind evtl Firmen auf diese Software angewiesen, und würden diese gerne weiterverwenden. Wenn der Quellcode verfügbar ist, so sind diese Firmen in der Lage die Software selber weiterzupflegen. Bzw können sich Firmen auf den Support solcher Software spezialisieren.

Alte Hardware, bei der nur Treiber für ältere Betriebssysteme verfügbar sind, deren Treiber können zu neuen Platformen portiert werden. So kann auch diese Hardware weiterverwendet werden und eine Neuanschaffung ist nicht nötig.

Zudem kann die Software dann akademisch analysiert werden
