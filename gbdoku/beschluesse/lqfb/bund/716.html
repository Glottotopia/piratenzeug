Feature Request

Der Bundesparteitag möge beschließen:

In der Software "Liquid Feedback" ist zu implementieren, dass einem Nutzer die Mitgliedschaft in einem Themenbereich entzogen wird, wenn dieser innerhalb von 12 Monaten keine der folgenden Aktivitäten in diesem Themenbereich getätigt hat:

  • Ein neues Thema im Themenbereich anlegen
  • Eine Delegation für ein Thema im Themenbereich festlegen
  • Eine alternative Initiative hinzufügen
  • Eine Initiative unterstützen
  • Eine Anregung schreiben oder bewerten
  • Sich an einer Abstimmung an einem Thema innerhalb des Bereiches beteiligen (kein Auto-Ablehnen)

Wird einem Nutzer die Themenbereichs-Mitgliedschaft entzogen, so ist dieser per Email an die im LiquidFeedback-System hinterlegte Email-Adresse zu informieren. Der betroffene Nutzer kann nach diesem Mitgliedschafts-Entzug jederzeit wieder ohne sonstige Beschränkungen Mitglied im Themenbereich werden (die 12-Monats-Frist beginnt dann von Neuem)

Begründung

Es besteht die Gefahr, dass viele Piraten sich in der Anfangsphase anmelden und in einigen Themenbereichen Mitglied werden; danach allerdings nie wieder im System arbeiten. Diese "passiven Mitglieder" erhöhen unnötig die Grundgesamtheit und damit die Hürde für notwendige Quoren. Es besteht die Gefahr, dass durch solche "permanent uninteressierten Mitglieder" das System nicht mehr sinnvoll genutzt werden kann.

Anmerkung zur Gegeninitiative

Es ist nicht meine Absicht, Mitglieder auszuschließen, die sich bewusst dafür entschieden haben, in allen Themenbereichen Mitglied zu sein und "Auto Ablehnen" aktiv zu haben. Es spricht aber nichts dagegen, dass man von einem solch passiven Mitglied verlangen kann, diesen Wunsch innerhalb einer Frist zu erneuern, wenn man keinerlei Eigenschaften eines in einem Themenbereich interessierten Mitglieds zeigt.

Wenn man Aktivität benötigt, um Initiativen zu unterstützen, so sollte auch Aktivität notwendig sein, wenn man Initiativen generell ablehnen möchte. Dieser Feature Request ist als Zusatz zum automatischen Löschen von Globaldelegationen bei Inaktivität zu betrachten.
